Gym Finder new

ExpressJs ReactJs Redux NodeJs Google Maps API Yelp API Heroku Webpack

Gym Finder is an app that helps you find gyms near your location or a location of your choice. It utilizes the Google Maps Autocomplete to find a location and uses the Yelp API to find gyms near you. It also uses the Google Maps Distance Matrix to find the distance between a gym and you. It is an open source project so any one who wish to create their own finder could simply change a line of code in the yelp API file.
